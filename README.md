# cppThermalCamera

read the thermal data from a topdon TC001 thermal camera

![output](.git-assets/capture-0.png)


# Controls

```shell
keymap:
     c  | toggle crosshair
     w  | toggle temp conversion
     h  | toggle hud
    z x | scale image + -
     m  | cycle through Colormaps
     p  | save frame to file
    r t | record / stop
     q  | quit

```
